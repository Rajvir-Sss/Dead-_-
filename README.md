## ⬇️ Installation

### Step 1: Download the Extension
1. Download the complete `SukNeo` folder
2. Keep all files and folder structure intact

### Step 2: Load in Chrome
1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in top-right corner)
3. Click **Load unpacked**
4. Select the `SukNeo` folder you downloaded
5. The extension icon should appear in your browser toolbar

### Step 3: Login
1. Click on the SukNeo extension icon
2. Star The Repo And Login with your credentials
3. You're ready to use all companion features!
